# Mimizuku

## Requirements
* PHP 5.6+
* WP-CLI
* Composer
* Node.js

## Browser support
* IE10 + Modern browser

## Get Started

### Build theme
```
$ yarn install
$ composer install
$ yarn run build
```

### Start built-in server
```
$ composer server
```
